# Twitter-like-char-counter

受験の小論文、志望理由書、各種テキスト編集用に作成した、ブラウザ上で動作するシンプルな文字数カウントツールです。

## 主な機能
* **超過部分のハイライト**: 設定した「文字数の上限」を超えた部分のテキスト背景に自動で色がつき、一目で文字数オーバーが分かります。
* **柔軟なカウント設定**: 改行や空白を文字数に「含める」「含めない」をチェックボックスでいつでも切り替えられます。
* **完全オフライン動作**: 入力したテキストは外部のサーバーに送信されず、すべてブラウザ内で完結するため、機密性の高い文章でも安心して編集できます。

## 使い方
1. リポジトリ内にある `Twitter-like-char-counter.html` をPCにダウンロードします。
2. ダウンロードしたファイルをダブルクリックしてブラウザ（Chrome, Edgeなど）で開きます。
3. 「文字数の上限」を入力（または400字、800字などのプリセットボタンをクリック）し、テキストエリアに文章を貼り付けて使用します。
